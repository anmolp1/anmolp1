# GitHub Profile Badges & Achievements Guide

A comprehensive guide to all GitHub profile badges, their requirements, and a plan to earn them.

---

## Obtainable Achievement Badges

| Badge | How to Earn | Tiers (Default / Bronze / Silver / Gold) | Difficulty |
|---|---|---|---|
| **Pull Shark** | Get pull requests merged | 2 / 16 / 128 / 1,024 PRs | Easy to start |
| **YOLO** | Merge your own PR without code review | 1 (no tiers) | Very Easy |
| **Quickdraw** | Close an issue or PR within 5 min of opening | 1 (no tiers) | Very Easy |
| **Pair Extraordinaire** | Co-author commits on a merged PR | 1 / 10 / 24 / 48 co-authored commits | Easy |
| **Galaxy Brain** | Get accepted answers in GitHub Discussions | 2 / 8 / 16 / 32 accepted answers | Medium |
| **Starstruck** | Have a repo earn stars | 16 / 128 / 512 / 4,096 stars | Hard |
| **Public Sponsor** | Sponsor someone via GitHub Sponsors | 1 (no tiers) | Easy (costs $) |

## Highlight Badges (Profile Labels)

| Badge | How to Earn |
|---|---|
| **Pro** | Subscribe to GitHub Pro ($4/mo) |
| **Developer Program Member** | Join the [GitHub Developer Program](https://github.com/developer) (free) |
| **Security Advisory Credit** | Submit an accepted advisory to the GitHub Advisory Database |
| **Security Bug Bounty Hunter** | Report a security vulnerability to GitHub |
| **GitHub Campus Expert** | Apply and join the GitHub Campus Program (students) |

## Action Plan

### Quick Wins (Day 1)
- [ ] **Quickdraw** — Open an issue, close it within 5 minutes
- [ ] **YOLO** — Open a PR, merge without review
- [ ] **Developer Program Member** — Join at github.com/developer (free)

### Short-Term (Week 1)
- [ ] **Pull Shark** — Merge 2+ PRs
- [ ] **Pair Extraordinaire** — Add `Co-authored-by: Name <email>` to a commit in a merged PR
- [ ] **Public Sponsor** — Sponsor any open source dev (min $1/mo)

### Medium-Term (Month 1)
- [ ] **Galaxy Brain** — Answer questions in GitHub Discussions and get them accepted
- [ ] **Pro** — Subscribe to GitHub Pro ($4/mo)

### Long-Term (Ongoing)
- [ ] **Pull Shark Gold** — 1,024 merged PRs
- [ ] **Starstruck** — Build a repo that earns 16+ stars
- [ ] **Security Advisory Credit** — Find and report a real CVE
